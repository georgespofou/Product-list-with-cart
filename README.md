# Product-list-with-cart
Product List and Cart is a beginner-friendly project that demonstrates how a basic shopping cart system works. The application displays a list of products and allows users to add and remove items from a cart, update quantities, and view the total price.
# Product List and Cart

## Description
A simple application that allows users to browse products and manage a shopping cart.

## Features
- View a list of available products
- Add and remove items from the cart
- Update product quantities
- Automatically calculate total price

## Technologies Used
- . HTML, CSS, JavaScript


## Purpose
This project was created for learning and practice purposes.
